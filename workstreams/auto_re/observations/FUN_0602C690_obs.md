## Computation Summary

317-line function. Combines orientation (+0x120-+0x12C), track angles
(+0x60/+0x64), roll projection (+0x100/+0x104), and world coordinates
(+0x140/+0x144) through multiple fixed-point multiplications (dmuls.l).
Uses constants 0x00200000 and others for scaling.

The function computes some form of collision/force magnitude by cross-
multiplying the orientation state with track geometry. Output goes to
+0x140/+0x144 which are then read by FUN_0602CA84 (force accumulator).

## Other Observations

- +0x140 and +0x144 are unique every frame even when idle. The dual-position
  NOP test (cycle 23) confirmed these are NOT internal position — they match
  exactly between NOPped and baseline runs.
- The function reads +0xFC (accel delta) — the only pipeline function besides
  the speed writer that reads this field. It may use accel delta as part of
  the force magnitude computation (integrating previous frame's acceleration
  into the current frame's force estimate).
